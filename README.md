# BizPulse SA 💹

A financial dashboard for South African small businesses with an AI-powered report generator.

## Features
- Real-time revenue, expense & profit tracking
- Income vs Expenses bar chart
- AI business report generator
- Add & categorise transactions
- Monthly target tracking

## Tech Stack
HTML · CSS · JavaScript · Groq AI API
